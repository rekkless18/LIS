# 登录页面设计文档

## 1. 页面整体布局

### 1.1 页面结构

- **背景区域**：全屏背景
- **Logo区域**：顶部显示系统Logo，页面中央偏左
- **登录组件**：包含所有登录元素，页面中央偏右
- **表单区域**：在登录组件内，包含用户名、密码等输入字段
- **操作区域**：在登录组件内，包含登录按钮、忘记密码等操作
- **底部信息**：版权信息、技术支持等描述

## 4. 表单区域设计

### 4.1 用户名输入框
- **标签**：用户名
- **输入框类型**：文本输入框
- **占位符**：请输入用户名
- **前缀图标**：用户图标
- **验证规则**：
  - 必填项
  - 长度：3-20个字符
  - 支持字母、数字、下划线
- **错误提示**：用户名不能为空
- **聚焦效果**：边框颜色 #1890ff
- **输入状态**：实时验证格式

### 4.2 密码输入框
- **标签**：密码
- **输入框类型**：密码输入框（可切换显示）
- **占位符**：请输入密码
- **前缀图标**：锁图标
- **后缀图标**：眼睛图标（显示/隐藏密码）
- **验证规则**：
  - 必填项
  - 长度：6-20个字符
- **错误提示**：密码不能为空
- **聚焦效果**：边框颜色 #1890ff

## 5. 操作区域设计

### 5.1 登录按钮
- **类型**：主要按钮
- **文字**：登录 / Login
- **宽度**：100%（自适应卡片宽度）
- **高度**：40px
- **背景色**：#1890ff
- **文字颜色**：#ffffff
- **字体大小**：16px
- **字体粗细**：500
- **圆角**：4px
- **悬停效果**：背景色 #40a9ff
- **点击效果**：背景色 #096dd9
- **加载状态**：显示加载动画，文字变为"登录中..."
- **禁用状态**：背景色 #d9d9d9，cursor: not-allowed

### 5.2 忘记密码按钮
- **文字**：忘记密码？
- **字体大小**：14px
- **颜色**：#1890ff
- **位置**：登录按钮右侧
- **对齐方式**：右对齐
- **悬停效果**：下划线
- **功能**：弹出提通用提示框，内容“该功能还在完善中，请稍候再来”
