# 标题栏设计文档

- **该组件为全局通用组件非页面** 

## 1. 组件整体布局

### 1.1 组件结构

- **侧导航折叠展开按钮**：用于控制另一个全局组件侧导航折叠或展开
- **面包屑导航**：展示当前页面
- **用户头像**：展示默认的头像
- **当前用户名称**：展示当前登录用户账号名称
- **用户下拉选项**：包含修改密码、退出登录选项

### 1.2组件位置

- **位于页面最上方**
- **组件上方边界与页面最上方一致**
- **组件右侧边界与页面最右侧一致**
- **组件左侧边界与侧导航右侧边界一致**：侧导航宽度为200px（展开状态）和64px（收起状态）
- **组件高度**：64px

## 2. 侧导航折叠展开按钮设计

### 2.1 侧导航折叠展开按钮样式

- **使用ant design的<MenuFoldOutlined />图标用于展示侧导航折叠展开状态,点击后折叠侧导航**
- **使用ant design的<MenuUnfoldOutlined />图标用于展示侧导航折叠折叠状态，点击后展开侧导航**
- **按钮位于标题栏左侧区域，距离标题栏组件最左侧边界12px**

## 3.面包屑导航设计

### 3.1 面包屑导航样式

- **使用import { Breadcrumb } from "antd";**
- **位置**：按钮位于标题栏左侧区域，位于侧导航折叠展开按钮右侧，距离侧导航折叠展开按钮12px
- **内容**：展示最多3级面包屑导航


## 4. 用户头像设计

### 4.1 用户头像样式

- **使用ant design的<Avatar />组件展示默认头像**
- **头像大小**：40px
- **位置**：标题栏右侧区域，位于当前用户名称左侧，距离当前用户名称12px

## 5. 当前用户名称设计

### 5.1 当前用户名称样式

- **字体大小**：14px
- **位置**：标题栏右侧区域，距离标题栏组件最右侧边界12px

## 6.用户下拉选项设计

### 6.1 用户下拉选项样式
- **触发区域**：光标点击用户头像和用户名的组合区域
- **关闭条件**：光标移出下拉选项区域或光标点击下拉选项
- **下拉选项**：重置密码；退出登录
  - 点击重置密码：弹出提通用提示框，内容“该功能还在完善中，请稍候再来”
  - 点击退出登录：返回登陆页面
- **位置**：下拉选项位于用户头像和用户名的组合区域下方
