# 首页页面设计文档

- **仅描述内容区，侧导航和顶标题均使用通用组件**
- **需要加载测试侧导航和顶标题**

## 1. 页面整体布局

### 1.1 页面结构

- **快捷入口区**：主要功能入口
- **待审批事项区**：位于页面下方左侧，显示待审批事项数量
- **实验室状态区**：位于页面下方右侧，显示实验室环境状态
- **设备告警区**：位于页面下方右侧，显示设备运行状态

## 2. 快捷入口区设计

### 2.1 快捷入口样式

- **位置**：位于页面上方，横向排列四个主要功能入口
- **内容**：每个入口为一个卡片，包含图标和标题
 - 订单查询:点击后跳转至订单查询页面
 - 样本查询:点击后跳转至样本查询页面
 - 物流查询:点击后跳转至物流查询页面
 - 报告查询:点击后跳转至报告查询页面

## 3. 待审批事项区设计

### 3.1 待审批事项样式

- **位置**：位于页面下方，左侧
- **内容**：卡片
  - 显示待审批事项数量，默认显示3
  - 显示即将超时文字和数量，默认显示1
- **跳转**：点击后跳转至审批查询页面

## 4. 实验室状态区设计

### 4.1 实验室状态区样式

- **位置**：位于页面下方，右侧偏上，位于设备告警区上方
- **内容**：卡片
  - 显示运行状态，默认显示运行正常
  - 显示环境参数
    - 温度，默认显示，26.3°
    - 湿度，默认显示，51.6%
- **跳转**：点击后跳转至环境管理页面

## 5. 设备告警区设计

### 5.1 设备告警区样式

- **位置**：位于页面下方，右侧偏下，位于实验室状态区下方
- **内容**：卡片
  - 显示设备状态
    - 冰箱：默认显示运行正常
    - AGV：默认显示故障
    - 测序仪：默认显示运行正常
    - 生化检测仪：默认显示运行正常
    - 血液检测仪：默认显示离线
- **跳转**：点击后跳转至设备管理页面
