# 检测项配置页面设计文档

- 仅描述内容区，侧导航和顶标题均使用通用组件
- 需要加载侧导航和顶标题

## 1. 页面整体布局

### 1.1 页面结构

产品配置页面采用标准列表页面布局：
- **查询条件区**：顶部卡片，可展开收起，包含多维度检索条件
- **功能按钮区**：查询条件下方，包含检测项配置操作按钮
- **数据列表区**：主要内容区域，显示检测项配置列表
- **分页控件**：底部，包含分页信息和每页条数选择

### 1.2 布局规格

- **查询条件区**：默认收起，展开时高度自适应
- **功能按钮区**：高度 64px，内边距 16px 24px
- **数据列表区**：最小高度 400px，自适应剩余空间
- **分页控件**：高度 56px，内边距 16px
- **响应式间距**：16px（移动端），24px（桌面端）

## 2. 查询条件区设计

### 2.1 查询条件样式

- **位置**：内容区最上方容器，和功能按钮区在同一个大的容器中的上方
- **展开/收起**：图标位于查询条件区右下，支持展开收起查询条件，默认收起，至少展示第一行
- **内容**
  - 文本：描述查询内容
  - 查询框：文本输入框、单选下拉框、日期选择器、多选下拉框、多选框（点击后弹窗）
- **布局**：采用自适应布局，宽度最小40px，每行最多四个查询条件，超过了换行

### 2.2 查询条件字段

#### 检测项编码

- **文本**：检测项编码
- **查询框类型**：文本输入框
- **占位符**：多个检测项编码用英文逗号分隔
- **支持格式**：多个检测项编码用英文逗号拼接查询，精确查询
- **验证规则**：支持字母、数字、特殊字符

#### 检测项名称

- **文本**：检测项名称
- **查询框类型**：文本输入框
- **占位符**：请输入检测项名称
- **查询方式**：模糊查询
- **验证规则**：支持中文、英文、数字

#### 检测项类型

- **文本**：检测项类型
- **查询框类型**：多选下拉框 
- **选项**：普检检测项、特检检测项、质谱检测项、研发检测项、其他检测项
- **默认状态**：全选
- **交互**：支持全选/取消全选

#### 结果判断类型

- **文本**：结果判断类型
- **查询框类型**：多选下拉框 
- **选项**：上限、下限、上下限、定性、阴阳性、聚合
- **默认状态**：全选
- **交互**：支持全选/取消全选

#### 状态

- **文本**：状态
- **查询框类型**：多选下拉框
- **选项**：启用、禁用
- **默认状态**：全选
- **交互**：支持全选/取消全选

#### 创建日期

- **文本**：创建日期
- **查询框类型**：日期选择器
- **占位符**：请选择创建日期
- **查询方式**：范围查询

## 3. 功能按钮区设计

### 3.1 功能按钮区样式

- **位置**：内容区最上方容器，和功能按钮区在同一个大的容器中的下方
- **展开/收起**：无展开折叠
- **内容**
  - 文本：描述查按钮显示内容
- **布局**：采用自适应布局，每行最多八个按钮，宽度最小24px，超过了换行
- **按钮位置**：
  - 左侧，按从左到右顺序：新建按钮，编辑按钮，删除按钮，启用按钮，禁用按钮
  - 右侧，按从右到左顺序：重置按钮，查询按钮

### 3.2 按钮字段

#### 新建按钮

- **文字**：新建
- **功能**：点击后弹出新建检测项弹窗
- **新建检测项弹窗规格**：
  - 标题：左上，显示"新建检测项"
  - 文本输入框：请输入检测项编码，必填
  - 文本输入框：请输入检测项名称，必填
  - 单选下拉框：请选择检测项类型，必填
    - 可选：普检检测项、特检检测项、质谱检测项、研发检测项、其他检测项
  - 单选下拉框：请选择结果判断类型，必填
    - 可选：上限、下限、上下限、定性、阴阳性、聚合
  - 检测项组件，根据结果判断类型选中的类型不同，组件内容不同：
    - 第一类（上限）
      - 文本输入框：请输入上限值框，支持输入数字，非必填
      - 文本输入框：请输入单位
    - 第二类（下限）
      - 文本输入框：请输入下限值框，支持输入数字，非必填
      - 文本输入框：请输入单位
    - 第三类（上下限）
      - 文本输入框：请输入上限值框，支持输入数字，非必填
      - 文本输入框：请输入下限值框，支持输入数字，非必填
      - 文本输入框：请输入单位
    - 第四类（定性、阴阳性）
      - 文本输入框：请输入结果值，支持输入中英文数字特殊符号，必填
    - 第五类（聚合）
      - 判断条件下拉框，选项：任意满足，所有满足，任意不满足，所有不满足
      - 选择检测项下拉选项+判断结果下拉框 组件，至少有一行，每一行是其他检测项，每个选择检测项下拉选项+判断结果下拉框右侧有按钮支持新增删除行
        - 选择检测项下拉选，选项：来源于检测项配置页面启用的检测项，必选
        - 判断结果下拉框，选项：偏高、偏低、正常、异常，必选
  - 按钮：提交（右下），关闭（右上角icon）

#### 编辑按钮

- **文字**：编辑
- **启用条件**：必须且只能勾选一个检测项
- **禁用状态**：置灰显示
- **禁用提示**：鼠标悬停时显示：请选择一个检测项
- **功能**：点击后弹出检测项编辑页面
  - 标题：左上，显示"编辑检测项"
  - 文本输入框：请输入检测项编码，必填
  - 文本输入框：请输入检测项名称，必填
  - 单选下拉框：请选择检测项类型，必填
    - 可选：普检检测项、特检检测项、质谱检测项、研发检测项、其他检测项
  - 单选下拉框：请选择结果判断类型，必填
    - 可选：上限、下限、上下限、定性、阴阳性、聚合
  - 检测项组件，根据结果判断类型选中的类型不同，组件内容不同：
    - 第一类（上限）
      - 文本输入框：请输入上限值框，支持输入数字，非必填
      - 文本输入框：请输入单位
    - 第二类（下限）
      - 文本输入框：请输入下限值框，支持输入数字，非必填
      - 文本输入框：请输入单位
    - 第三类（上下限）
      - 文本输入框：请输入上限值框，支持输入数字，非必填
      - 文本输入框：请输入下限值框，支持输入数字，非必填
      - 文本输入框：请输入单位
    - 第四类（定性、阴阳性）
      - 文本输入框：请输入结果值，支持输入中英文数字特殊符号，必填
    - 第五类（聚合）
      - 判断条件下拉框，选项：任意满足，所有满足，任意不满足，所有不满足
      - 选择检测项下拉选项+判断结果下拉框 组件，至少有一行，每一行是其他检测项，每个选择检测项下拉选项+判断结果下拉框右侧有按钮支持新增删除行
        - 选择检测项下拉选，选项：来源于检测项配置页面启用的检测项，必选
        - 判断结果下拉框，选项：偏高、偏低、正常、异常，必选
  - 按钮：提交（右下），关闭（右上角icon）

#### 删除按钮

- **文字**：删除
- **启用条件**：必须至少勾选一个检测项
- **禁用状态**：置灰显示
- **禁用提示**：鼠标悬停时显示：请至少选择一个检测项
- **功能**：点击后弹出删除确认弹窗
- **删除确认弹窗规格**：  
  - 标题：左上，显示"删除检测项"
  - 文字：确定要删除选中的检测项么？操作后不可恢复
  - 按钮：确定，关闭（右上角icon）

#### 启用按钮

- **文字**：启用
- **启用条件**：至少勾选一个检测项且状态为"禁用"
- **禁用状态**：置灰显示
- **禁用提示**：鼠标悬停时显示：请至少选择一个禁用检测项
- **功能**：点击将选中检测项状态改为"启用"

#### 禁用按钮

- **文字**：禁用
- **启用条件**：至少勾选一个检测项且状态为"启用"
- **禁用状态**：置灰显示
- **禁用提示**：鼠标悬停时显示：请至少选择一个启用检测项
- **功能**：点击将选中检测项状态改为"禁用"

#### 查询按钮

- **文字**：查询
- **功能**：点击后按查询条件查询检测项

#### 重置按钮

- **文字**：重置
- **功能**：点击后将所有查询条件重置为默认值并进行一次查询

## 5. 分页控件设计

### 5.1 分页控件样式

- **位置**：内容区下方左侧

### 5.2 分页控件字段

- **第一页**：双左箭头图标
- **上一页**：左箭头图标
- **页码输入**：可输入页码，Enter 跳转
- **下一页**：右箭头图标
- **末页**：双右箭头图标
- **总条数**：显示"共 {total} 条"
- **当前页**：显示"第 {current} 页"
- **每页条数**：下拉框，选项：10、20、50、100
